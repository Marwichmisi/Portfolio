Comment connecter n8n à Telegram

Recevoir des notifications sur son téléphone quand un formulaire est rempli ou qu'un serveur tombe en panne est crucial. Telegram est l'outil parfait pour cela grâce à son API gratuite et rapide.

1. Créer le Bot avec BotFather

Tout commence dans l'application Telegram :

  1. Cherchez l'utilisateur @BotFather (le bot certifié).
  2. Envoyez la commande /newbot.
  3. Donnez un nom à votre bot (ex: MonAssistant_Bot).
  4. Important : Copiez le Token API qui apparaît (il ressemble à 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11).

2. Configurer les Identifiants dans n8n

Ouvrez votre instance n8n :

Allez dans Credentials > New > cherchez "Telegram API".

Collez simplement votre Token API récupéré à l'étape précédente. Donnez-lui un nom clair (ex: "Mon Bot Perso").

3. Trouver votre Chat ID

Pour que le bot sache à qui envoyer le message, il lui faut un ID.

  • Démarrez une conversation avec votre nouveau bot sur Telegram (cliquez sur "Start").
  • Envoyez-lui un message quelconque ("Hello").
  • Dans n8n, utilisez le nœud Telegram Trigger pour écouter ce message. L'ID apparaîtra dans les données JSON (sous chat.id).

4. Le Nœud Telegram en Action

Vous pouvez maintenant ajouter un nœud Telegram à la fin de n'importe quel workflow.

Sélectionnez l'opération Send Message, entrez votre Chat ID, et tapez votre texte dynamique. C'est aussi simple que ça !
