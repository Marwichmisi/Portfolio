Contrôler un servomoteur avec Arduino Uno

Le servomoteur est l'actionneur roi de la robotique. Contrairement à un moteur classique qui tourne en continu, le servo peut maintenir une position précise (entre 0 et 180 degrés). Idéal pour un bras robotique ou... une barrière de péage !

1. Le Matériel Nécessaire

Arduino Uno Servomoteur SG90 3x Jumper Wires

2. Le Câblage (Wiring)

Le servo a 3 fils, attention à ne pas les inverser :

  • Marron (ou Noir) : GND (Masse) → Connecter au GND Arduino.
  • Rouge : VCC (+) → Connecter au 5V Arduino.
  • Orange (ou Jaune) : Signal (PWM) → Connecter au Pin 9.

3. Le Code C++

Nous utilisons la bibliothèque standard Servo.h qui gère les signaux PWM pour nous.

#include <Servo.h>

Servo monServo;  // Création de l'objet servo

void setup() {
  monServo.attach(9);  // On attache le servo au Pin 9
}

void loop() {
  // Aller à 0 degré
  monServo.write(0);
  delay(1000);

  // Aller à 90 degrés
  monServo.write(90);
  delay(1000);

  // Aller à 180 degrés
  monServo.write(180);
  delay(1000);
}

4. Pour aller plus loin

Ce code fait faire des allers-retours au moteur. Pour un projet réel, vous pourriez contrôler cette position en fonction d'un capteur de distance (comme dans mon projet de péage) ou d'un potentiomètre.
